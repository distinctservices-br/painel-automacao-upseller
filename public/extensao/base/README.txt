INSTALAÇÃO — UPSELLER SYNC (5 passos, uma única vez)

1. Extraia esta pasta do arquivo zip
2. Abra o Chrome e acesse: chrome://extensions
3. Ative o "Modo do desenvolvedor" (canto superior direito)
4. Clique em "Carregar sem compactação"
5. Selecione a pasta extraída

Pronto! O ícone aparecerá na barra do Chrome.
Faça login no Upseller normalmente — a sincronização é automática
e invisível.
